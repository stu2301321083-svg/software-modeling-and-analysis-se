## Project Description
RealEswipe is an academic project focused on database design, modeling, and data analysis.
The project represents a conceptual real estate platform for apartment listings, bookings,
matches, and reviews.

The main focus of the project is:
- Conceptual and logical database modeling
- Entity-Relationship diagrams
- Data Warehouse design
- Analytical reporting with Power BI

No executable application is implemented. The project is intended for analysis,
design validation, and visualization purposes.

## Database Logic Description

The database design includes:
- User management
- Apartment listings
- Booking and matching logic
- Review system
- Moderation process

Logical descriptions of:
- Stored procedure: `real_eswipe.sp_create_booking`
- Function: `real_eswipe.dates_overlap`
- Trigger: `trg_update_match_status_after_booking`

are provided as part of the database design documentation.
